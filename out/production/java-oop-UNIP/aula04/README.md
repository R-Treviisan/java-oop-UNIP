Exercícios 1 a 3
📌 Descrição

Os exercícios 1 ao 3 da Aula 4 foram desenvolvidos apenas com base no enunciado proposto em sala.

🎯 Objetivo

Esses exercícios têm como foco a prática de estruturas de repetição (loops simples), com o objetivo de:

Compreender o funcionamento de laços de repetição

Aplicar estruturas como for, while ou do-while

Exercitar lógica básica de controle de fluxo

🛠 Observação

Por se tratarem de exercícios introdutórios sobre controle de loop simples, não foi necessária implementação avançada ou divisão em múltiplas camadas (como entidades ou classes auxiliares).
