# Aula 02 - Exercício 01

Este diretório contém duas implementações do mesmo exercício, com níveis diferentes de estruturação.

## 📁 exercicio01_enunciado

Implementação direta conforme solicitado no enunciado do professor.

Características:
- Estrutura mais simples
- Menor separação de responsabilidades
- Foco na execução básica do problema

## 📁 exercicio01_completo

Implementação estruturada aplicando boas práticas de Programação Orientada a Objetos.

Características:
- Separação em pacotes (application / entities)
- Encapsulamento
- Métodos organizados por responsabilidade
- Código mais preparado para manutenção e expansão

---

Objetivo: comparar a solução básica com uma versão estruturada seguindo princípios de organização e modelagem.
