📘 Projeto Aula 03 – Programação Orientada a Objetos
📌 Descrição

Este repositório contém os exercícios desenvolvidos na Aula 03 da disciplina de Programação Orientada a Objetos.

O objetivo é aplicar conceitos fundamentais de orientação a objetos em Java, incluindo:

Encapsulamento

Associação entre classes

Construtores

Métodos de acesso (getters e setters)

Sobrescrita de toString()

Organização em pacotes

🛠 Tecnologias Utilizadas

Java 17+

IntelliJ IDEA

Paradigma Orientado a Objetos

🎯 Objetivos de Aprendizado

Modelagem de classes

Relacionamento entre objetos (Associação)

Manipulação de dados via Scanner

Boas práticas de organização de código

Separação entre camada de aplicação e entidades

▶ Como Executar

Abra o projeto na IDE (IntelliJ).

Navegue até o pacote application.

Execute a classe Program.

📚 Conceitos Aplicados

Associação entre objetos (ex: Funcionario → Holerite)

Encapsulamento de atributos

Uso de construtores parametrizados

Sobrescrita de métodos padrão
